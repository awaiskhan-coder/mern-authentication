# MERN Authentication System
A full-stack MERN Authentication System built using React, Node.js, Express.js,
and MongoDB. The application provides secure user authentication using JWT,
password encryption with bcrypt, profile management, and password update functionality.
The project is fully deployed using Vercel (Frontend) and Railway (Backend).

## 🌐 Live Demo

**Live Website:** https://mern-authentication-three.vercel.app

## 💻 GitHub Repository

**Repository:** https://github.com/awaiskhan-coder/mern-authentication

## ✨ Features

- 🔐 Secure User Authentication using JWT
- 🔑 Password Hashing with bcrypt
- 👤 User Registration and Login
- 📝 Update User Profile
- 🔒 Change Password
- 🛡 Protected Routes
- 📱 Responsive User Interface
- 🌐 RESTful API Integration
- ☁️ MongoDB Atlas Database
- 🚀 Deployed on Vercel and Railway

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React.js | Frontend UI |
| Vite | Frontend Build Tool |
| Tailwind CSS | Styling |
| Axios | API Requests |
| Node.js | Runtime Environment |
| Express.js | Backend Framework |
| MongoDB Atlas | Database |
| Mongoose | MongoDB ODM |
| JWT | Authentication |
| bcrypt | Password Hashing |
| Railway | Backend Deployment |
| Vercel | Frontend Deployment |
